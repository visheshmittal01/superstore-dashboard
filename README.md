# Superstore Sales Dashboard using POWER BI
### Project Title
<b>Superstore Sales Dashboard</b>

### Project Overview
The Superstore Sales Dashboard is an interactive data visualization project designed to provide actionable insights into global sales performance. This project transforms raw transactional data into a high-level executive summary, allowing stakeholders to track key performance indicators (KPIs), regional trends, and category-wise revenue distribution.

### Purpose
The goal of this project is to analyze sales data to identify:
•	<b>Regional Performance:</b> Understanding which continents drive the most volume.<br>
•	<b>Customer Behavior:</b> Identifying preferred payment methods and high-demand product categories.<br>
•	<b>Revenue Trends:</b> Monitoring how revenue shifts across different product segments.<br>
•	<b>Efficiency Metrics:</b> Tracking average sales and transaction volume to gauge business health.

### Tools Used
•	<b>Power BI Desktop:</b> Power BI Desktop: Used for the overall layout, design, and interactivity of the dashboard.<br>
•	<b>Power Query Editor:</b> Utilized for ETL (Extract, Transform, Load) processes, data cleaning, and transformation to ensure data accuracy.<br>
•	<b>DAX:</b> Used to create complex measures and calculated columns for custom metrics (e.g., Total Sales, Average Sales).


### Features and Highlights
• <b>Dynamic KPI Cards:</b> prominent display of critical metrics including Total Sales (₹392.22K), Quantity Sold (2K), Transactions (500), and Average Sales (₹784.43).<br>
• <b>Geospatial Analysis:</b> A "Total Sales by Region" map that visually highlights revenue hotspots across continents.<br>
• <b>Payment Method Breakdown:</b> A donut chart analyzing the distribution of transactions via Cash, PayPal, Bank Transfer, and Credit Card.<br>
• <b>Category Performance:</b>
  • <b>Funnel/Bar Chart:</b> Ranks product categories (Beauty, Electronics, Home, etc.) by total sales volume.<br>
  • <b>Line Chart:</b> Tracks total revenue trends across different categories to identify high-growth areas.
